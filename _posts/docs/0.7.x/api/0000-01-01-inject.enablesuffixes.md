{% highlight js %}
Inject.enableSuffixes();
{% endhighlight %}

Enables the auto-appending of a ".js" extension to module paths. This is the default behavior.

See: [Inject.disableSuffixes()](/docs/0.7.x/api/inject.disablesuffixes.html)

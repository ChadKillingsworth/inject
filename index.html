Inject is the link that brings CommonJS to the Browser. Inject is a fast and lightweight library that removes the pain of dependencies in the browser. If you've written modules for node.js, then you're equipped to write code for the browser using Inject. Our mission is to make managing JavaScript dependencies painless.

  • CommonJS Compliant
  • Support for AMD Modules
  • Works Cross-Domain
  • Advanced caching model
